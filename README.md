# fullstack-app
Stack:
  - React
  - Express
  - Postgres

To run this application you need two terminal instances.
You need to have NodeJs and Npm installed
In root folder run:
  
  1.
  # yarn
  # or
  # npm install
  And then
  2.
  # node server.js

  Afterwards go to *client* folder and repeat first step.
  And then run
  3.
  # yarn start or npm  start
